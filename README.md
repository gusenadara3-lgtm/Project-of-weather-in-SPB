## Описание проекта
В этом проекте мы реализовали систему прогнозирования и анализа погоды по CSV‑файлу.
В файле WEATHER_SPB.csv хранится температура за каждый день годов 2025-2026: дата и числовое значение температуры. Программа читает этот файл, считает разные статистики и показывает работу нескольких структур данных: префиксные суммы, бинарное дерево поиска, очередь и стек.

## Структура проекта
В корне репозитория находятся три файла:
```text
weather_project/
├── main.py
├── WEATHER_SPB.csv
└── README.md

**Назначение файлов:**
* `main.py` — основной файл программы, содержащий логику обработки данных, структуру классов и демонстрационные алгоритмы;
* `WEATHER_SPB.csv` — файл базы данных с ежедневными записями полей `date` и `temp`;
* `README.md` — структурированное описание проекта, функционала и использованных алгоритмов;
* `.gitignore` — список файлов и папок, которые не должны попадать в репозиторий.
